# 📦 SourceDump

**SourceDump** — это легковесная Python-утилита, которая превращает ваш проект в единый, структурированный текстовый дамп. Идеально подходит для передачи контекста всего репозитория в ChatGPT, Claude или другие LLM за один раз.

## ✨ Основные возможности

* **Tree + Content**: Генерирует визуальное дерево проекта и собирает содержимое всех файлов.
* **Smart Filtering**: Исключает бинарные файлы, системные папки (`.git`, `__pycache__`) и всё лишнее по умолчанию.
* **Гибкие исключения**: Поддержка флага `-e` и файлов исключений в стиле `.gitignore` (через `-ef`).
* **Token Counter**: Точный подсчет токенов (через `tiktoken`), чтобы вы знали, влезет ли код в контекстное окно нейросети.
* **Zero-Dependency**: Работает на чистом Python (библиотека `tiktoken` опциональна для точности).

## 🚀 Быстрый старт

### Простой дамп текущей папки
```bash
python prompt_packer.py -o context.md
```

### Использование с файлом исключений
```bash
python prompt_packer.py -ef .gitignore -o context.md
```

### Выборочный дамп конкретных модулей
```bash
python prompt_packer.py ./src ./internal -e "*.log" "tests/" -o project.txt
```

## 🛠 Установка

1. Скачайте скрипт `prompt_packer.py`.
2. (Опционально) Установите библиотеку для точного подсчета токенов:
```bash
pip install tiktoken
```

## 📖 Параметры запуска

| Флаг | Полное имя | Описание |
| :--- | :--- | :--- |
| `paths` | — | Пути к папкам для анализа (по умолчанию `.`) |
| `-o` | `--output` | Путь к файлу результата (если нет — вывод в консоль) |
| `-ef` | `--exclude-file` | Файл с паттернами исключений (например, `.gitignore`) |
| `-e` | `--exclude` | Дополнительные паттерны исключений через пробел |

---

## 📝 Формат исключений (exclude_file.txt)
Скрипт поддерживает как построчную запись, так и перечисление в ряд:
```text
.git
__*
node_modules
dist
secrets.json
temp/*.tmp
```

---

## 💡 Почему это удобно?
Вместо того чтобы копировать файлы по одному, вы скармливаете нейросети один файл. Благодаря XML-подобной разметке `<file path="...">...</file>`, современные модели понимают структуру проекта и связи между модулями.
