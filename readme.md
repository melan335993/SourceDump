# 📦 SourceDump

**SourceDump** — это легковесная Python-утилита для превращения вашего проекта в структурированный текстовый дамп. Она создана, чтобы «скармливать» контекст репозитория в LLM за один клик.

## ✨ Основные возможности

* **Tree + Content**: Наглядное дерево проекта и содержимое файлов в XML-тегах `<file>`, которые идеально распознаются современными нейросетями.
* **Smart Filtering**: Умный пропуск бинарных файлов и системных папок (`.git`, `__pycache__`, `node_modules`, `uv.lock`) по умолчанию.
* **Режим Diff (`--diff`)**: Добавляет системную инструкцию, заставляющую ИИ отвечать только изменениями (Unified Diff), а не переписывать файлы целиком.
* **Оптимизация токенов (`--strip`)**: Удаление пустых строк из исходного кода, что позволяет уместить в промпт на 15–25% больше полезной информации.
* **Гибкий экспорт**: Поддержка флага `-e` и файлов исключений (`-ef`) в стиле `.gitignore`. Скрипт понимает как построчную запись, так и перечисление паттернов в ряд.
* **Token Counter**: Безопасный и точный подсчет токенов через `tiktoken` (с обработкой спецтокенов типа `<|endoftext|>`).

## 🚀 Быстрый старт

### Максимально упакованный контекст для правок
Сэкономит токены и настроит ИИ на генерацию патчей (диффов):
```bash
uv run SourceDump.py --strip --diff -o prompt.md
```

### Использование с файлом исключений
```bash
uv run SourceDump.py -ef .gitignore -o context.md
```

### Специфический выбор папок и паттернов
```bash
uv run SourceDump.py ./src ./internal -e "*.log" "out/" "ncad_samples" -o project.txt
```

## 📖 Параметры запуска

| Флаг | Полное имя | Описание |
| :--- | :--- | :--- |
| `paths` | — | Пути к папкам для анализа (по умолчанию `.`). |
| `-o` | `--output` | Путь к файлу результата (если нет — вывод в консоль). |
| `-ef` | `--exclude-file` | Файл с паттернами исключений (поддерживает формат `.gitignore`). |
| `-e` | `--exclude` | Дополнительные паттерны исключений через пробел. |
| `--diff` | — | Добавляет в начало дампа инструкцию для ИИ: «отвечай только диффами». |
| `--strip` | — | Удаляет все пустые строки из кода для экономии контекстного окна. |

## 📝 Формат файла исключений
Скрипт автоматически распознает оба формата записи в файлах `-ef`:

**Вариант А (построчно):**
```text
.git
__*
node_modules
My Projects
```

**Вариант Б (в ряд):**
```text
.* __* node_modules build bin *.exe *.jpg
```
